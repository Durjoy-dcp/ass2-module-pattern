# ass2-module-pattern

## Installation

Create .env in the ./ directory and insert variables there

```
PORT=
DATABASE_URL=
B_Salt_round=
```

To run the src file as development enviornment :

```bash
npm run start:dev
```

To run the dist file as production enviornment :

```bash
npm run start:prod
```

Live site : https://ass2-module-pattern.vercel.app/
